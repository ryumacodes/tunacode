The Rules

Application Layers: ui → core → tools | indexing | lsp | infrastructure → utils
Enforced top-to-bottom. No upward imports.
Foundation: types and configuration
Exempt from layer rules. Any module can import them directly. They import nothing (or only each other).
✓ Allowed: ui → core, core → tools, core → types, utils → configuration
✗ Forbidden: utils → core, tools → ui, types → utils, configuration → core
Layer 2 Modules:
  • tools/ - Tool implementations (bash, read_file, etc.) + tools/messaging/ facade + tools/parsing/
  • indexing/ - Code indexing functionality
  • lsp/ - Language Server Protocol
  • infrastructure/ - File filtering, system utilities
